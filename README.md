# User Balance Manager

A front-end web application for managing user balances and tracking transactions. This project provides a simple and interactive interface for users to view and update their account balances.

##  Tech Stack 
- **HTML** – structure and layout
- **CSS** – styling and responsive design
- **JavaScript** – interactive functionality

##  Features 
- Display user balances
- Add or subtract amounts from user accounts
- Track basic transaction history (front-end only)
- Responsive and interactive interface
